HTML

HTML stands for HyperText Markup Language. It's the standard language used to create web pages. It uses tags to structure the content and define its appearance. You can use HTML to create headings, paragraphs, links, images, and more!

CSS

CSS, short for Cascading Style Sheets, is a language used to style and format web pages. It allows you to control the appearance of HTML elements, such as fonts, colors, layouts, With CSS, you can make your web pages look visually organized.

JS

JavaScript, often abbreviated as JS, is a programming language used to make web pages interactive. It allows you to add functionality to your website, such as form validation, dynamic content, and interactive elements

SQL

SQL, which stands for Structured Query Language, is a programming language used for managing and manipulating databases. It allows you to perform tasks like querying, inserting, updating, and deleting data in a database.
